# AWS Glue ETL Boilerplate Docs

## Scope

This boilerplate provides a generic Medallion-style AWS Glue ETL foundation.

## Included guides

- `DEVELOPMENT.md`: local/dev workflow
- `ENVIRONMENT_VARIABLES.md`: runtime configuration
- `ARCHITECTURE.md`: high-level project architecture
- `TESTING.md`: testing conventions
- `MIGRATION_GUIDE.md`: move private sources to boilerplate patterns
